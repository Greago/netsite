Network Basics

Communication Model

  • Source:
    • -generates data to be transmitted
  • Transmitter:
    • -converts data to transmittable signals.
  • Transmission system:
    • -carries data
  • Reciever:
    • -converts recieved signal to data.
  • Destination:
    • -Takes incoming data.

    Transmission modes

    There are three modes of data communication namely:
    • Simplex:
      • -refers to communication in only one direction e.g. one way radio and television broadcast does not allow the listener to communicate back through the reciever.
    • Half Duplex:
      • -Refers to cmmunication in both direction but one direction at a time e.g. the two way radio(walkie talkie).
    • Full duplex transmission:
      • -Refers to communication in both directions simultaneously e.g. computers sending and recieving data on a network

    Types Of Computer Networks

    Computer networks can be classified using different characteristics and asprcts. However, the common classifications are:
    According to geographical size:
    • Local Area Network(LAN):
      • -covers small geographical area .
    • Metroplitan Area Network(MAN):
      • -covers a town or city(metropolitan area).
    • Wide Area Network(WAN):
      • -covers more than one metropolitan area, country or the entire globe. An example is the internet.

    According to communication model applied:
    • Point-to-point networks:
      • -the data is sent from one computer to the next through direct links without broadcasting.
    • Broadcast networks:
      • -a message is broadcast on the network and is visible to all computers on the network. The destination network/computer picks the message from the medium.
